6.2 Example 2: Stacking Many Plots Together – Multiplot

In this example, we use the multiplot environment to produce a gallery of several plots. The set origin command is used to position each one. We also make use of multiple -axes in the top-left plot: the functions and are plotting together, but on different scales. The output is shown in figure 6.2.
