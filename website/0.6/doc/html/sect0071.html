: Command Reference : set : noarrow

5.21.24 noarrow

set noarrow [<arrow number>]

Issuing the command: 

set noarrow

removes all arrows, as set using the set arrow command, from the current plot. Alternatively, individual arrows can be removed using the syntax: 

set noarrow 2

where the tag ‘2’ here is the identification number given to the arrow to be removed when it was initially set using the set arrow command.
