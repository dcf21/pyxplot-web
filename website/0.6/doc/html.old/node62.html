5.10 move 

move <plot number> <x>, <y>

The move command is part of the multiplot environment; it can be used to move items around on a multiplot page. The desired item to be moved should be identified using the reference number which it was given when it was created; it would have been displayed on the terminal at that time. For example: 

move 23 8,8

If 23 were a plot, this would move it to position 8,8 (measured in centimetres). The end result would be the same as if the command set origin 8,8 had been called before plotting plot number 23.
