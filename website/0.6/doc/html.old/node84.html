5.21.11 fontsize 

set fontsize <value>

The fontsize setting changes the size of the fount5.1 used to render all text labels which appear on a plot, including keys, axis labels, etc. The value specified should be an integer in the range -4 to 5, corresponding to LATEX's tiny (-4) and Huge (5) sizes, for example: 

set fontsize 2

The default value is zero, LATEX's normal fount size. As an alternative, fount sizes can be specified directly in the LATEXtext of labels, for example: 

set xlabel '\Large This is a BIG label'
