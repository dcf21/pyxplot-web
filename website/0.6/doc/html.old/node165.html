6.7 Example 7: Bar Charts - Box Widths

In this example, we demonstrate different ways of specifying the widths of bars on a bar chart. In panel (a), the widths are automatically determined from the data, changing bar midway between datapoints. In panel (b), the wboxes plot style is used, which reads the widths of the bars from a third column in the datafile. In panel (c), we demonstrate how the set boxfrom command can be applied to bar charts, as well as to impulses. And in panel (d) we illustrate how the fillcolour modifier can be used to produce coloured bars. The output is shown in figure 6.7.
