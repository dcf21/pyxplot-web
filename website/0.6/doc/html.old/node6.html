1.3.1 Installation as User

The following steps describe the installation of PyXPlot from a .tar.gz archive by a user without superuser (i.e. root) access to his machine. It is assumed that the packages listed above have already been installed; if they are not, you need to contact your system administrator.

  • Unpack the distributed .tar.gz: 

    tar xvfz pyxplot_0.6.1.tar.gz
cd pyxplot

  • Run the installation script: 

    ./configure
make

  • Finally, start PyXPlot: 

    ./pyxplot
