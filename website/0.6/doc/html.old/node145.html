5.21.51 xlabel 

set xlabel '<text>'

The xlabel setting controls the label placed on its -axis (abscissa). For example: 

set xlabel '$x$'

sets the label on the -axis to `'. Labels can be placed on higher axes by inserting their number after the `x', for example: 

set x10label 'foo'

would label the tenth axis.

Similarly, labels can be placed on -axes as follows: 

set ylabel '$y$' 
set y2label 'foo'
