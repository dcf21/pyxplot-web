1.3.1 Installation within Linux Distributions

PyXPlot is available as a user-installable package within some Linux distributions. Gentoo†See http://gentoo-portage.com/sci-visualization/pyxplot and Ubuntu†Note that there is an error in the packaging of PyXPlot in Hardy Herron and Intrepid Ibex, which means that the tetex-extra package, upon which PyXPlot depends, is not automatically installed with PyXPlot. already have such packages. Debian will have such a package in their next release, Lenny, which is scheduled for late 2008. Alternatively, and to ensure that they are using the latest version, Debian and Ubuntu users can download the package from the PyXPlot website and install it manually by typing: 

dpkg -i pyxplot_0.7.0.deb
