Pyxplot User's Manual: latest version

The users' manual for the latest version of Pyxplot (0.9.3) is available for free download under the GNU General Public License (GPL) Version 2. It is available as a single pdf download, or can be browsed online.

There are some minor incompatibilities between Pyxplot 0.8.x and 0.9.x. For more information about how to convert scripts, see the full changelog.

Older releases

Manuals are also remain available for the following historical releases of Pyxplot. However, we no longer offer any support for these versions and do not recommend their use for new projects.
