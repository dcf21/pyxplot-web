5.3 Including Data within Command Scripts

It is also possible to embed data directly within Pyxplot scripts, which may be useful when a small number of markers are wanted at particular pre-defined positions on a graph, or when it is desirable to roll a Pyxplot script and the data it takes into a single file for easy storage or transmission.
