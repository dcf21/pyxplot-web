1.17 exit

exit

The exit command can be used to quit PyXPlot. If multiple command files, or a mixture of command files and interactive sessions, were specified on PyXPlotâ€™s command line, then PyXPlot moves onto the next command-line item after receiving the exit command.

PyXPlot may also be quit be pressing CTRL-D or using the quit command. In interactive mode, CTRL-C terminates the current command, if one is running. When running a script, CTRL-C terminates execution of the script.
