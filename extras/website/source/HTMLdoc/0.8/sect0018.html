3.15 Interactive Help

In addition to this Users’ Guide, PyXPlot also has a help command, which provides a hierarchical source of information. Typing help alone gives a brief introduction to the help system, as well as a list of topics on which help is available. To display help on any given topic, type help followed by the name of the topic. For example, 

help datafile

provides information on the format in which PyXPlot expects to read datafiles and 

help plot

provides information on the plot command. Some topics have sub-topics, which are listed at the end of each page. To view them, add further words to the end of your help request – an example might be 

help set title
