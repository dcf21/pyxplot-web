1.46.5 axisunitstyle

set axisunitstyle ( bracketed | squarebracketed | ratio )

The setting axisunitstyle controls the style with which the units of plotted quantities are indicated on the axes of plots. The bracketed option causes the units to be placed in parentheses following the axis labels, whilst the squareÂ­bracketed option using square brackets instead. The ratio option causes the units to follow the label as a divisor so as to leave the quantity dimensionless.
