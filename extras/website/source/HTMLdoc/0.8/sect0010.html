2.2.2 Installation as User

The following steps describe the installation of PyXPlot from a .tar.gz archive by a user without superuser (i.e.Â root) access to his machine. The required archive can be downloaded from the PyXPlot website. It is assumed that the packages listed above have already been installed; if they are not, you will need to contact your system administrator.

  • Unpack the distributed .tar.gz: 

    tar xvfz pyxplot_0.8.4.tar.gz
cd pyxplot

  • Run the installation script: 

    ./configure
make

  • Finally, start PyXPlot: 

    ./bin/pyxplot
