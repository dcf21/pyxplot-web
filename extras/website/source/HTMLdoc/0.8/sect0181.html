1.46.57 papersize

set papersize ( <named size> | <height>,<width> )

The setting papersize changes the size of output produced by the postscript terminal, and whenever the enlarge terminal option is set (see the set terminal command). This can take the form of either a recognised paper size name – a list of these is given in Appendix 5 – or as a (height, width) pair of values, both measured in millimetres. The following examples demonstrate these possibilities: 

set papersize a4
set papersize letter
set papersize 200,100
