1.1 PyXPlot’s Heritage: Gnuplot

PyXPlot’s interface bears some striking similarities to that of Gnuplot. Specifically, the commands used for plotting simple graphs in the two programs are virtually identical, though the syntax used for more advanced plotting often differs and PyXPlot’s mathematical environment is hugely extended over that of Gnuplot. This means that Gnuplot users will have a head start with PyXPlot: simple Gnuplot scripts will often run in PyXPlot with minimal modification.
