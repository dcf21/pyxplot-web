8 Configuring PyXPlot

In Parts I and II, we encountered numerous configuration options within PyXPlot which can controlled using the set command. There are times, however, when many plots are wanted in a homogeneous style, or when a single plot is repeatedly generated, when it is desirable to change the default set of configuration options with which PyXPlot starts up, in order to avoid having to repeated enter a large number of set commands. In this chapter, we describe the use of configuration files to program PyXPlot’s default state.
